## Setup and Run:
### ***Follow these instructions without skipping any steps. Skipping any Steps may lead to the server setup failing***

- ### *Install Docker and Docker-compose on your System*
- ### *Create a file in the root of the project named ```.env```* 
- ### *Paste the Contents of ```sample.env``` in ```.env``` and modify the contents accordingly*
- ### *docker-compose build*
- ### *docker-compose up*
- ### *Access the Server at* 
        http://localhost:8085/
